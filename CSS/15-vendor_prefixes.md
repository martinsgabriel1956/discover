# Vendor Prefixes

Permite que browsers adicione `features` a fim em uso de colocar alguma novidade que vemos no CSS:

```CSS
  -webkit-background-clip: text; /* Chrome, Safari, IOS e Android */
  -moz-background-clip: text; /* Firefox */
  -ms-background-clip: text; /* Internet Explorer */
  -o-background-clip: text; /* Opera */
```

Em outras palavras, serve para converter novas funcionalidades do CSS em navegadores ou vers√µes de navegadores mais antigos.
