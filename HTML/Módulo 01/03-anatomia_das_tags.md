# Anatomia das Tags

```HTML
<!--Elemento-->
  <!--Abertura de Tag-->
    <h1>
    <!--Conteúdo-->
      Hi
    </h1>
  <!--Fechamento de Tag-->
<!--/Elemento-->
```

Elementos vázios === Não tem como adicionar conteúdo, não fecha a tag e depende dos atributos para o seu funcionamento

```HTML
<!--Tag Vazia-->
  <!--Atributos-->
<img 
  src="" 
  alt=""
>
<!--/Tag Vazia-->
```
