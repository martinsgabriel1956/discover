# Utilizando a tag âncora

```HTML

      <ul>
        <li>
          <a href="#history">História</a>
        </li>

        <li>
          <a href="#resume">Resumo</a>
        </li>

        <li>
          <a href="#about">Sobre</a>
        </li>
      </ul>
      <h1>
        Trabalho
      </h1>
```
